# Обработчик табличных данных (Data_Cleansing)

**Подготовка данных — это 80% времени работы над любой аналитической задачей или ML-моделью.** Часто сырые датасеты содержат ошибки и пропуски, которые мешают дальнейшему анализу и обучению алгоритмов. 

Этот проект был создан для того, чтобы автоматизировать рутину Exploratory Data Analysis (EDA) и предварительной очистки данных. Он представляет собой удобное Desktop-приложение с графическим интерфейсом для анализа, визуализации и обработки таблиц в формате CSV. Инструмент позволяет быстро находить аномалии и устранять дефекты в датасетах без необходимости писать скрипты для каждой новой таблицы, делая данные готовыми к работе в несколько кликов.

##  Технологический стек
* **Язык:** Python 3.14
* **Интерфейс:** Tkinter (ttk)
* **Анализ данных:** Pandas, NumPy
* **Визуализация:** Matplotlib

##  Ключевой функционал

*  **Диагностика данных:** Интерактивная таблица с отображением типов данных и подсчетом пропущенных значений (NaN). Столбцы с пропусками автоматически подсвечиваются красным индикатором.
*  **Умная визуализация:** * Для числовых признаков: автоматическое построение гистограмм с линиями среднего арифметического и медианы.
  * Для категориальных (текстовых) признаков: столбчатая диаграмма частотности (Топ-10 значений).
*  **Методы очистки (Imputation):** Заполнение пропусков с помощью статистических метрик (Медиана, Среднее, Мода), замена на константу или полное удаление строк с пропусками.
*  **Массовая обработка:** Функция «Заполнить всё» для автоматической очистки всей таблицы в один клик (медиана для чисел, мода для категорий).
*  **Система откатов:** Встроенная история действий с возможностью многоуровневой отмены (до 15 шагов назад) благодаря структуре данных `collections.deque`.
*  **Гибкий импорт/экспорт:** Поддержка кодировок `utf-8` и `cp1251`, автоматическое определение разделителей (`Авто`, `;`, `:`, `Tab`), сохранение чистого датасета в новый CSV-файл.

##  Как запустить локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/kepchupchik/Data_Cleansing.git
   ```
2. Перейдите в папку с проектом и установите необходимые библиотеки:
   ```
    pip install -r requirements.txt
    ```

3. Запустите основное приложение:

```
python main.py
```
